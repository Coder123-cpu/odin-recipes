# Recipes Website

## Introduction

This project is a simple recipe website built entirely with **HTML**. It consists of multiple web pages that showcase different recipes, each containing ingredients, preparation steps, and images where applicable. The goal of this project is to strengthen my understanding of the fundamental concepts of web development by creating a structured, accessible, and easy-to-navigate website without using CSS or JavaScript.

## Skills Demonstrated

By completing this project, I will have demonstrated my ability to:

* Create well-structured HTML documents.
* Use semantic HTML elements to improve readability and accessibility.
* Build and organize a multi-page website.
* Create internal links for seamless navigation between pages.
* Add images, lists, headings, paragraphs, and other common HTML elements.
* Organize project files and directories effectively.
* Use Git and GitHub to manage version control and publish a project.

This project serves as a foundation for future web development projects that will incorporate CSS for styling and JavaScript for interactivity.
